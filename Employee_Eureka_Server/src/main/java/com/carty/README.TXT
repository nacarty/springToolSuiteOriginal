SPRING MICROSERVICES
===================
SEE HTTPS://JAVAINUSE.COM/SPRING/SPRING-CLOUD-NETFLIX-ZUUL-TUTORIAL
for this precise example.


This Micro-Services Project consists of 3 Spring Boot Projects:
1. Employee_Zuul
2. EmployeeConsumer
3. EmployeeProducer
4. Employee_Eureka_Server

To run, start them as Spring Boot applications in this order:
1. Employee_Zuul 
2. EmployeeProducer
3. Employee_Zuul
4. EmployeeConsumer
